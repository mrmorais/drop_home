About Drop

Drop is a open blogging platform in development by the community. We want to create a easier way of write and share knowledges at Web.

It have no owner, and everyone can use and change the Drop. You can help us build Drop interacting with our community, building code, discussing improments, and sharing what is Drop.

Our community is so young! If you are starting with open source, please join us.

We are on Gitter.im, and our code is on GitHub. There are some instructions about how to start to contribute at our GitHub repository.
